# Lunara — Telegram-бот

## Flow

1. `/start` → onboarding (имя, дата, время, место) → создаётся **карта** в `charts`
2. PNG карта + бесплатный разбор (на карту)
3. Paywall Premium (249 ₽) — на аккаунт Telegram
4. **🌙 Мои карты** — список и выбор активной карты
5. **➕ Создать новую карту** — ещё один профиль

Гороскопы, вопросы и совместимость работают от **активной карты**.
Premium и покупки вопросов/гороскопов — на **telegram_id** (аккаунт).

## Локально

```bash
pip install -r requirements.txt
cp .env.example .env   # BOT_TOKEN, OPENAI_API_KEY
python3 bot.py
```

## Railway

Variables: `BOT_TOKEN`, `OPENAI_API_KEY`

Start: `python bot.py`

**Только один экземпляр бота** (Railway ИЛИ локально).
